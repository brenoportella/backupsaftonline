# Backup SaftOnline
A webscrapy to store all the information available on saftonline

### Setup
**clone the repository**
```bash
git clone https://github.com/brenoportella/backupsaftonline
cd backupsaftonline
```

**install requirements**
```bash
pip install -r requiriments.txt
```

### Usage
```bash
python backup.py
```
